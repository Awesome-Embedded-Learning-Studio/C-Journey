# 进阶项目：从零写一个操作系统

> ⚠️ **超出 ROADMAP 8 阶段主线的进阶实践**。建议在完成 [阶段 5 系统编程](../../ROADMAP.md) 后作为综合挑战。
> 📖 先读导览：[从零写一个操作系统：通电到内核的接力赛](../../documents/advanced/0-os-from-scratch.md)

从引导扇区到内核线程与文件系统，按“课次”循序渐进地实现一个 x86 裸机操作系统。

## 组织方式

- `1/` ~ `20/`、`15+/` — 按课次递进的实现（MBR → loader → 中断 → 内存 → 线程 → …）
- `final/` — 阶段性完整版，含清晰的分层结构：

```text
final/
├── boot/      # MBR.S / loader.S 引导
├── kernel/    # init / interrupt / memory / debug
├── thread/    # 线程与同步（sync.c / switch.S）
├── fs/        # 文件系统
└── command/   # shell 命令
```

- `ready/` — 准备性练习（如纯汇编 hello world）

## 如何使用

- 用 `nasm` 汇编 `.S` / `.s`，用 `dd` 写入软盘 / 硬盘镜像
- 在 **Bochs**（见各目录 `bochsrc`）中运行调试
- 每个课次目录通常自带 `Makefile`

## 学习要点

- 裸机编程：启动文件、链接脚本、中断向量表、`volatile` 寄存器
- 从 MBR 到加载内核的完整引导链
- 分页内存、内核线程、同步原语的最简实现

> 这是一个长期演进的项目，部分课次为实验性代码。阅读 `final/` 可获得最完整的视图。
