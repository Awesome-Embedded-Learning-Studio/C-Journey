# 行为准则

参与 C-Journey 的每个人都承诺营造开放、友善、尊重的社区。

- **友善耐心**:欢迎各水平的学习者,从零基础到老手。
- **尊重差异**:不因经验、背景、身份而歧视或轻视。
- **就事论事**:对代码 / 内容的批评对事不对人。
- **接纳反馈**:被指出错误时理性回应,纠正就好。
- **乐于互助**:能回答新手问题就回答,营造「折腾工程师同行者」的氛围。

## 不可接受的行为

骚扰、人身攻击、仇恨言论、持续性打扰,或其他让他人感到不适的行为。维护者有权关闭、移除此类言论或当事人。

## 反馈

遇到违反准则的行为,请通过 GitHub 联系仓库维护者(私下报告)。

本准则受 [Contributor Covenant](https://www.contributor-covenant.org/) 启发。
