# 示例代码 (examples/)

配合 [ROADMAP](https://awesome-embedded-learning-studio.github.io/C-Journey/roadmap) 各阶段的小型可运行示例。建议先阅读对应阶段的 `documents/`，再到这里动手实践。

| 目录 | 阶段 | 主题 |
|---|---|---|
| [stage0-compiling-and-debug](./stage0-compiling-and-debug/) | 阶段 0 | 编译流程、GDB 调试、动 / 静态库 |
| [stage1-armc-basics](./stage1-armc-basics/) | 阶段 1–2 | 变量、堆、位运算 |
| [stage4-cmake-lib](./stage4-cmake-lib/) | 阶段 4 | CMake 库工程：静态/动态库 + install/export/find_package |
| [stage5-tcp-socket](./stage5-tcp-socket/) | 阶段 5 | TCP socket 编程 |

> [`hello.c`](./hello.c) 是阶段 0 的入门示例。
